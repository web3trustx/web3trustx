Frequently Asked Questions

Do I need an API key to use TrustXData?

For basic use in demo mode (up to 100 requests/hour), no key needed. For regular use, sign up for free and get 1,000 requests/hour.

Is the data real-time?

Yes, our data is updated every 30 seconds from multiple blockchains and verified sources.

Which blockchains do you support?

We currently support BNB Chain and Solana. Coming soon: Ethereum, Polygon, Base, and Sui.

Can I use the API commercially?

Yes, all our plans allow commercial use. Review the terms of service for more details.

Do you offer technical support?

Free Plan: email support. Pro Plan: priority support. Enterprise: dedicated 24/7 support.
